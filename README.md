# Computer Graphics Project

The Walter's destiny

Lazar Savić, april 2024

## Key Bindings

* `WASD` - movement
* `R` - spotlight on/off
* `B` - Blinn-Phong on/off
* `H` - HDR on/off
* `SPACE` - bloom on/off
* `Q` - decrease exposure
* `E` - increase exposure
* `ESC` - terminates the program
